Порядок настройки компьютера:
1) скачать компилятор python
2) скачать pip, если python 3.4 и выше, то pip установливается с ним
3) добавить путь pip в переменные среды, зайти в папку c:/user/user/local/approaming/python/scripts
этот путь надо добавить в переменные среды через панель управления - система и безопасность - система- доп параметры системы - добавить в path
4) скачать те компоненты которые нужны
pip install имя_модуля

Настройка сервера
1) на машине-сервере необходимо отредактировать два файла
pg_hba.conf нужно добавить внизу: host all all all md5
postgresql.conf нужно изменить строку listen_adresses = "*"
2) добавить исключение для порта 5432 в настройках брандмауэра, для входящих подключений добавить порт 5432
3) создать сервер с адресом local host

1) на удаленной машине зайти в pgadmin4, добавить сервер, имя любое, ip адрес указываем машины-сервера, порт 5432, maintenance database указываем БД к которой подключимся, пользователя можно оставить postgres и пароль для указываем, какой был при установке
